# Derecho Artificial - Perspectivas

Proyecto editorial de análisis jurídico de IA dirigido por Ricardo Scarpa. Análisis crítico e independiente sobre la intersección del Derecho, la Ética y la Inteligencia Artificial.

## Tecnologías utilizadas

This project is built with:
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Markdown + JSON for content (in /content/ and /public/Recursos/)
- Deployed automatically on Vercel via GitHub push

## Desarrollo Local

Para trabajar localmente en este proyecto, asegúrate de tener instalado Node.js y npm.

1. **Clonar el repositorio**
   ```sh
   git clone https://github.com/artificialderecho-ux/derecho-artificial-perspectivas.git
   ```

2. **Instalar dependencias**
   ```sh
   npm install
   ```

3. **Iniciar el servidor de desarrollo**
   ```sh
   npm run dev
   ```

## Despliegue

Deployment: Push to main branch → Vercel builds and deploys automatically (connected via GitHub integration).
